# Backend FF Imports

## Instalação

Clone o projeto

```bash
git clone https://github.com/SEU_USUARIO/catalogo-ff-backend.git
```

Instale as dependências

```bash
npm install
```

Copie

```
.env.example
```

para

```
.env
```

Preencha

```
PORT

IGEST_API_URL

IGEST_TOKEN

IGEST_COMPANY_ID
```

Execute

```bash
npm run dev
```

Servidor

```
http://localhost:3000
```
